## Description
This project is created as part of the 42 curriculum. The goal is to sort a stack, I have 2 stacks (a and b) and a set of instructions to manipulate both stacks.
libft and printf were included in this project.

## Instructions

### Installation
To download the project to your machine, follow these steps:

1.  Open your terminal in your preferred directory.
2.  Clone the repository:
    ```bash
	git clone <paste_your_ssh_link_here>
	```

### Compilation
The project includes a Makefile. You can compile the library using:
```bash
make
```

This will generate the push_swap.exe. The program will show instructions needed to sort the stack.

### How to Use
```bash
make
./push_swap.exe <your input>
```
You can insert the input with or without double quotes.
